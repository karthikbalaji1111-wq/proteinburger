# Protein Burgers

The website lives in **`protein-burgers/`**. You must run the dev server — opening `index.html` directly will show a blank page.

## Run the site

```bash
cd /Users/karthikbalaji/Documents/proteinburger/protein-burgers
npm install
npm run dev
```

Your browser should open automatically. If not, use the URL printed in the terminal (often `http://localhost:5173/`).

From the parent folder you can also run:

```bash
cd /Users/karthikbalaji/Documents/proteinburger
npm run dev
```

## Pages

- Home: `http://localhost:5173/`
- Menu: `http://localhost:5173/menu`

## If the page is blank

1. Confirm the terminal shows **VITE ready** and a Local URL.
2. Open that exact URL (port may be 5174 or 5175 if 5173 is busy).
3. Do not double-click `index.html` — use `npm run dev` only.
4. Hard refresh: **Cmd + Shift + R** (Mac).
